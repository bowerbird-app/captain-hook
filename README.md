# CaptainHook

This repository is a template for building Rails mountable engine gems.

For instructions on how to use, configure, and rename this template, please see the [Gem Template README](docs/captain_hook/CAPTAIN_HOOK_README.md).

## Important: After Renaming

When you rename this gem using `bin/rename_gem`, the documentation in `docs/captain_hook/` is intentionally **preserved** as architectural reference material. These files remain unchanged to document the original template structure and serve as a reference for the patterns used in this gem.
